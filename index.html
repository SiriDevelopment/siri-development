Siri Features

  • Entertain you with Siri's fun commands.
  • Moderates your server with our awsome moderation commands.
  • Lots of utility commands for you guys to use.

Add to server

Command Description Example
botinfo Shows bot info j!botinfo
cat Shows a random pic of a cat j!cat
confesschannel Setups a channel for anonymous messages. Then get your server id and send the bot a direct message with the prefix confess [server id] ex: confess 25309857029789 test j!confesschannel #channel
dog Shows a random pic of a dog j!dog
emojis Shows a list of all custom server emojis j!emojis
help Shows all commands j!help
ping Checks if Bot is online j!ping
react [emoji] Sends a reaction j!react gifcolorheart
reacttolast [emoji] Reacts to the last message with the given emoji j!reacttolast gifcolorheart
reacttothis [emoji] Reacts to the message with the given emoji j!reacttothis gifcolorheart
say [message] Lets the bot say something j!say hello
serverinfo Shows server info j!serverinfo
userinfo [@user] Shows user info j!userinfo
j!userinfo Peter
ytsearch [youtube-title] Searches for videos on YouTube j!ytsearch blackpink

linklog [logType] [#channel]
    (admin) Links a channel for logging
  • modlog: logs used mod commands
  • username: logs username changes
  • nickname: logs nickname changes
  • deleted: logs deleted messages
  • edited: logs edited messages
 p!link modlog #modlog
p!link deleted #deletedMessages
linkrole [roleType] [@role] (admin) Links a role
Role types: modrole		 p!linkrole modrole @mod
unlink [logType/roleType] (admin) Unlinks a channel or role
Log Types: modlog, username, nickname, deleted, edited Role Types: modrole 		p!unlink modlog
p!unlink modrole
